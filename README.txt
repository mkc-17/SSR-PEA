This is a reference implementation of SSR-PEA as described in the paper:
A Search Space Reduction Based Progressive Evolutionary Algorithm for Influence Maximization in Social Networks. Lei Zhang, Kaicong Ma, Haipeng Yang, Cheng Zhang, Haiping Ma and Qi Liu. TCSS. 2022

Before to execute SSR-PEA, it is necessary to install the following packages:
pip install numpy
pip install networkx

The supported input format is an edgelist:
node1_id_int	node2_id_int
